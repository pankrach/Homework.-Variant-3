Домашняя работа по Алгоритмическим Языкам Понкратова С., ИУ8-24
Вариант 3 
Пояснение:
Совпадение с кодом Гончарова С. обусловлено тем, что мы проживаем в одной комнате, и другого способа выполнять домашнюю работу, кроме того, как совместно, мы не придумали.

Команды для работы

Команды добавления:
add work [имя сотрудника] [название подразделения]- добавление сотрудника в подрозделение
add sub [название подразделения 1] [название подразделения 2]- добавление подразделения 1 в подразделение 2
addsubd [имя сотрудника] [название подразделения] – добавление второй ставки сотруднику
Команды удаления:
del work [имя сотрудника] - увольнение сотрудника
del sub [название подразделения] - расформирование подразделения
delsubd [имя сотрудника] [название подразделения] –увольнение сотрудника со 2 ставки
Другие команды:
transfer  work [имя сотрудника] [название подразделения] – перевод сотрудника между подразделениями
transfer  sub [название перемещаемого подразделения] [название подразделения, куда перемещают]-перемещение подразделения 1 в подразделение 2
list -  вывод всех текущих данных программы 
inform [имя сотрудника] – поиск сотрудника и информация о нём
open – считывание данных из файла “data.txt” для дальнейшей работы с ними в консоли
save – сохранения текущих данных программы в файл “data.txt”
merger [имя подразделения] [имя подразделения] – объединение подразделений

Пример последовательности команд:

add sub s1 head
add sub s2 head
add sub s3 s1
add work w1 s1
add work w2 s2
add work w3 s3
list
transfer work w2 s1
list
transfer sub s2 s1
list
merger s3 s2
list
addsubd w2 s2
inform w2
delsubd w2 s2
inform
del work w3
list
del sub s2
list
save
